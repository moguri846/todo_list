# vue-todo

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

![Alt text](/img/vue-todo.png)

## 기능

>> todo 추가
>> todo 저장
>> todo 완료
>> todo 수정
>> todo 삭제
>> todo 모두 삭제

## vuex

vuex를 사용하여 상태관리 했습니다.
